Coincell CR2032 supported 3V.
SD Card supported Voltage 2.7V-3.6V.
Measuring behind diodes before SD Card 2.4V-2.8V.
EFM32G230 Microcontroller Operating supply voltage 1.85V-3.8V.
VBATT_2 supported 3.2V-3.6V.

Alternatives:
- Use another batteryholder for battery LS14250 1/2AA 3.6V for small packages.
- For long durations use 12V Lithium-Ion batteries. https://github.com/peterloes/Data_Logger/blob/main/Getting_Started_Tutorial/3_Supplier.txt

Power Pins BATT_1_INPUT and BATT_1_GND supported 10V-17V.
